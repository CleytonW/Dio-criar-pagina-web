<h1> até <h6>

(Voltar)

As tags h1 ao h6 são tags que utilizamos como cabeçalhos, ou seja, nossos titulos. Cada uma dessas tags tem um tamanho diferente, como o h1 sendo a maior e o h6 sendo o menor. Deixando bem claro que a tag h1 só pode ser utilizada uma vez em cada página, pois a mesma é considera como um título principal.

Elemento <p>

(Voltar)

O elemento HTML p, são tags que utilizamos para indentificar paragráfos. Esses paragráfos podem conter tantos texto como outros elementos de hypertexto(imagens, vídeos, links e etc.)

<Mark>

(Voltar)

O elemento <mark> é utilizado para quando queremos marcar/grifar ou deuxar um trecho em destaque.

<small>

Esté elemento ele funcio de um modo simples, deixando toda as letras dentro de sua tag com fonte menor.
